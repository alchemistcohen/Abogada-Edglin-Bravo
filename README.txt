# 🌐 Página Profesional - Abogada Edglin Bravo

Bienvenido al repositorio del sitio web profesional de la **Abogada Edglin Bravo**, especializada en derecho migratorio, civil y de familia, con amplia experiencia en el acompañamiento legal a migrantes hispanohablantes en España y América Latina.

## 🧩 Descripción

Este proyecto tiene como objetivo ofrecer una plataforma clara, moderna y funcional donde los clientes puedan:

- Conocer el perfil profesional de la abogada.
- Consultar servicios legales ofrecidos.
- Acceder a contenido informativo sobre temas jurídicos.
- Contactar directamente para agendar asesorías.

## ⚙️ Tecnologías utilizadas

- **HTML5, CSS3, JavaScript** – Estructura, estilos y funcionalidades base.
- **Bootstrap / TailwindCSS** – Framework CSS para diseño responsive .
- **React.js**  – Para una experiencia dinámica e interactiva.
- **GitHub** – Control de versiones y despliegue.
- **Hosting**: GitHub Pages
